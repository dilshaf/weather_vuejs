This template should help get you started developing with Vue 3 in Vite.

## Recommended IDE Setup

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (and disable Vetur) + [TypeScript Vue Plugin (Volar)](https://marketplace.visualstudio.com/items?itemName=Vue.vscode-typescript-vue-plugin).

## Customize configuration

See [Vite Configuration Reference](https://vitejs.dev/config/).

## Libraries Used
In this project, we use the following libraries:
1. Vue JS - The main library for building user interfaces
2. TailwindCSS-For Designing the project

## Project Explanation
Creating a weather app using Vue.js and Tailwind CSS involves several steps, from setting up the project to fetching weather data and displaying it in the user interface. API used
from OpenWeatherMap.In this project include error handling,javascript networking,fetching data.
